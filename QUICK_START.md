# Quick Start Checklist

Get your cybersecurity portfolio live in 30 minutes or less!

## ⚡ Quick Start (5-10 minutes)

- [ ] **Open your portfolio**: Double-click `index.html` to open in your browser
- [ ] **Preview it**: Take a minute to explore the site
- [ ] **Check all features**: Test the navigation, buttons, form, responsive design

## 📝 Customize Basics (10-15 minutes)

### Edit index.html:
1. **Find & Replace "Cybersecurity Student & Enthusiast"** with your headline
2. **Find & Replace "SecurePort"** with your name or brand (2 places)
3. **Update About section**:
   - Replace the About Me paragraph with your story
   - Update the 3 stats (Projects, Certifications, Months)
4. **Update Contact section**:
   - Replace email placeholder
   - Add your LinkedIn, GitHub, and other links

### Quick Edits (Search for these):
```
your.email@example.com        → Your actual email
SecurePort                     → Your name/brand (appears in navbar & footer)
your Name                      → Your actual name
yourprofile                    → Your GitHub/LinkedIn profile names
```

## 🚀 Deploy (5 minutes)

### Option 1: GitHub Pages (Recommended - Free)
1. Create account at [github.com](https://github.com)
2. Create new repository named `portfolio` (or any name)
3. Upload your 4 files:
   - index.html
   - styles.css
   - script.js
   - blog-post-template.html (optional)
4. Go to Settings → Pages
5. Enable GitHub Pages on main branch
6. Your site will be live at: `username.github.io/portfolio`

### Option 2: Netlify (Recommended - Even Easier)
1. Go to [netlify.com](https://netlify.com)
2. Drag & drop your project folder
3. Site is instantly live with a free URL
4. Get free custom domain later

### Option 3: Traditional Host
1. Upload files to your web hosting provider
2. Point your domain to the hosting
3. Access via your domain

## 🎨 Common Customizations

### Change Colors
Edit `styles.css`, find `:root {` section:
```css
--primary-color: #0066ff;   /* Main blue */
--accent: #00d9ff;          /* Cyan highlight */
--secondary-color: #ff6b35; /* Orange accent */
```

### Change Font
Add to HTML `<head>`:
```html
<link href="https://fonts.googleapis.com/css2?family=Poppins:wght@400;600;700&display=swap" rel="stylesheet">
```

Edit `styles.css`:
```css
body {
    font-family: 'Poppins', sans-serif;
}
```

### Add Logo/Image
Add to navbar or hero:
```html
<img src="your-logo.png" alt="Logo">
```

## 📞 Support Resources

### If something isn't working:

**Mobile menu not showing?**
- Check viewport meta tag is in HTML
- Test on actual phone or browser dev tools

**Colors showing wrong?**
- Clear browser cache (Ctrl+Shift+Delete)
- Try different browser

**Form not working?**
- Check browser console for errors
- Ensure form fields have correct IDs

**Links broken?**
- Verify file paths are correct
- Test with absolute vs relative paths

**Site loading slow?**
- Check image file sizes
- Consider optimizing with online tools

### For more help:
- Read README.md in your portfolio folder
- Check PORTFOLIO_GUIDE.md for detailed tips
- Visit [Stack Overflow](https://stackoverflow.com) for coding help
